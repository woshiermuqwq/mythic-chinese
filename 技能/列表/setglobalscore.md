技能: Set Global Score
--------------------------

设置全局记分板分数.

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| objective | obj, o  | 用于修改的记分板名,不存在则创建 |         |
| value     | v       | 修改值              
                                                                            
示例
----

    - setglobalscore
        {
        objective=someobjective;
        v=2
        } ~onAttack
