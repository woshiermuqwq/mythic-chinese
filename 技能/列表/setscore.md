技能: Set Score
--------------------------

修改虚拟目标在指定记分板上的分数（上限2147483647）.  
虚拟目标名为玩家名, 相当于修改指定玩家的分数.

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| objective | obj, o  | 用于修改的记分板名,不存在则创建 | 无 |
| value     | v       | 修改值 | 无 |
| name  | n, entry, e    | 虚拟玩家名                    | dummy   |
                                                                            
示例
----

```yaml
 Skills:
 - setscore{o=someobjective;v=2} @self ~onAttack
```

造成伤害后将虚拟玩家: dummy 在记分板: someobjective 的分数设为2.
