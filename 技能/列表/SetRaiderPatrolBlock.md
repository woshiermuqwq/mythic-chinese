技能: Set Raider Patrol Block
--------------------------

允许/禁止技能目标（掠夺者）加入巡逻队

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| bool | b, can, c | 为`true`时可进入, 反之不可加入 | true |

示例
--------

```yaml
 Skills:
 - setRaiderCanJoinRaid @mir{t=pillager;r=100} ~onTimer:200
 - ...
```


拓展信息
-------

- 别称: setCanJoinRaider
