技能: Base Damage
--------------------------

对技能目标造成等值于(手持武器攻击力 + 基础攻击力) * 百分比的伤害  
施法者为玩家时无效，变相实现方法见下.

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| multiplier           | m | 所乘以的百分比（从 5.2.0 起支持[占位符](/技能/占位符)）         | 1    |
| [Damage](/技能/列表/damage)除修改项: damage外的所有修改项 | | | |

示例
--------

受伤后对目标造成15[(0 + 10) * 1.5]伤害.

```yaml
测试:
 Type: HUSK
 Damage: 10
 Skills:
 - bd{m=1.5} @T ~onDamaged
```

右键后对目标造成(6x3)点伤害  
版本不低于 4.11 且需付费版

```yaml
测试:
 Id: diamond
 Lore:
 '- 物理伤害: 3'
 Skills:
 - damage{a=6* ％ap
 - _attack:max％} @T ~ouse
```

1 = 100%

拓展信息
--------

- 别称: bd, weapondamage, wd
