**描述:** 若实体名称为指定字符串.

---

**修改项:**

| 修改项名  | 别称           | 描述                      | 默认值 |
| --------- | -------------- | ------------------------- | ----- |
| name | n | 用于检测的名称 | Ashijin |

---

**示例:**

```
Conditions:
- name{n=Sharrk_kunjang}
```

---

**拓展信息:**

- [x] 检测对象: 实体
- [x] 别称: castername
