技能: Bar Remove
--------------------------

为目标移除Boss血条.

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| name      | n       | 要修改的血条内部ID.                                | infobar                     |

示例(写于实体配置内)
--------

```
    Skills:
    - barRemove{name="MyBossBar"} @self ~onInteract
    - ...
```
