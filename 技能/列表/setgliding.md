技能: Set Gliding
--------------------------

设置技能目标身上的鞘翅飞行状态.

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| gliding      | g       | 开关 | true      |

示例（实体配置）
--------

```yaml
 Skills:
 - setgliding{g=true} @self ~onDamaged
```
受伤后展开自己的鞘翅.
