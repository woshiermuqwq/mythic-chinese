技能: Bar Set
--------------------------

为目标修改Boss血条.

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| name      | n       | 要修改的血条内部ID.                                | infobar                     |
| display   | d       | 修改后的血条文本.                          | &lt;skill.var.aura-name&gt; |
| value     | v       | 修改后的血条百分比. | 1.0                         |

示例
--------

    Skills:
    - barSet{name="MyBossBar";display="<caster.name> - <caster.hp>";value=1.0} @self ~onDamaged
    - ...
