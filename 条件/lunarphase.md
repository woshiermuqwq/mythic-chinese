**描述:** 若所出世界的月相为.

---

**修改项:**

| 修改项  | 别称      | 描述                             |
| ---------- | ---------- | --------------------------------------- |
| phases     | p, phase   | 用于检测的月相数(可多个) |

---

注意,phrases的值必须是数字:

|   月相名    | Phase Number |
| --------------- | -------------|
| 满月       |       0      |
| 亏凸月  |       1      |
| 下弦月   |       2      |
| 残月 |       3      |
| 新月        |       4      |
| 娥眉月 |       5      |
| 上弦月   |       6      |
| 盈凸月  |       7      |

---

**示例:**

```
Conditions:
- lunarphase{p=0} true
```
```
Conditions:
- lunarphase{p=0,2,4,6} true
```

---

**拓展信息:**

- [x] 检测对象: 位置
