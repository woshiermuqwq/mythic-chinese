技能: Activate Spawner
--------------------------

激活指定名称的生成点(组).

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| spawners | spawner,s | 要激活的生成点(组) | 无 |

示例
--------

激活生成点"测试"
```yaml
 Skills:
 - activatespawner{spawner=测试}
```
激活位于"测试"组内的所有生成点
```yaml
 Skills:
 - activatespawner{spawner=g:测试}
```
激活所有以"测试"为前缀的生成点
```yaml
 Skills:
 - activatespawner{spawner=测试*}
 - ...
```
