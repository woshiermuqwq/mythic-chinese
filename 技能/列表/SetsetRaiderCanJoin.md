技能: set Raider Can Join
--------------------------

允许或禁止技能目标（掠夺者）加入一个灾厄巡逻队

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| bool | b, can, c | 为`true`时允许加入巡逻队, 反之禁止 | false |

示例
--------

```yaml
 Skills:
 - setcanjoinraid @self ~onSpawn
 - ...
```
施法者生成后无法加入灾厄巡逻队
