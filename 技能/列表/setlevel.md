技能: Set Level
--------------------------

修改技能目标的Mythic等级.  
修改后技能目标血量会回满.

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| action    | a       | 修改模式           | SET           |
| level     | l       | 修改值 | 1             |

示例（实体配置）
--------

生成后设置自身等级为3.
```yaml
 Skills:
 - setlevel{a=set;l=3} ~onSpawn
```
击杀玩家后升级.
```yaml
 Skills:
 - setlevel{a=add;l=1} ~onKillPlayer
```
额外信息
--

- [x] 别称: modifylevel
