技能: Set Raider Patrol Block
--------------------------

令技能目标（掠夺者）获得或取消灾厄巡逻队队长的身份

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| bool | b, can, c | 为`true`时将变为队长 | false |

示例
--------

```yaml
 Skills:
 - setRaiderLeader @self ~oninteract
 - ...
```
施法者被右键后将成为灾厄巡逻队队长

-------
