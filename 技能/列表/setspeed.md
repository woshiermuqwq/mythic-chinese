技能: Set Speed
--------------------------

设置目标指定移动方式下的移动速度.

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| speed      | 修改后的移动速度 | 1             |
| type       | 所修改的移动方式,WALKING(步行) 或 FLYING(飞行) | WALKING       |

示例 (实体配置)
--------
       
      Skills:
    - setspeed{speed=2;type=walking} ~onSpawn

生成后将自身步行状态下的移动速度设为2.

额外信息
-------

所修改的移动方式为Walking时, 可修改实体在水中的速度
