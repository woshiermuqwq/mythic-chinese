技能: Set Block Type
--------------------------

修改指定位置的方块种类.

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| material  | m       | 方块种类. | DIRT          |
| data      | md      | 方块特殊值.           | 0             |

示例
--------

    方块修改示例:
      Skills:
      - setblock{m=STONE;md=0} @selflocation
