技能: Track Location
--------------------------

记录目标的当前位置.

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| 无 | 无 | 无 | 无 |

新增于MM 4.13

示例
--------

      Skills:
      - tracklocation @trigger ~onInteract
      - teleport @trackedlocation ~onDamaged

右键后记录右键者的位置, 受伤后传送到该位置.
