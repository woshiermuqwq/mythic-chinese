**描述:** 若与实体的出生点距离在指定范围之内.

---

**修改项:**

| 修改项名  | 别称           | 描述                      |
| --------- | -------------- | ------------------------- |
| distance | d | 用于检测的数值范围 |

---

**示例:**

```
Conditions:
- distancefromspawn{d=<100} true
```

```
Conditions:
- distancefromspawn{d=>50} true
```

---

**拓展信息:**

- [x] 检测对象: 位置
- [x] 别称:
