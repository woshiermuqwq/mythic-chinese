技能: Set Mob Color
--------------------------

设置目标(特定实体)的颜色.

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| color | c | 颜色名 | 无 |

示例  (实体配置)
--------
    
    Skills:
    `- setcolor{color=blue} @self ~onSpawn`

生成后修改自身颜色为蓝色
