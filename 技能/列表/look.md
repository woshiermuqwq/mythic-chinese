技能: Look
--------------------------

看向目标.  
实体 **必须** 拥有AI.

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| headOnly    |         | 是否只有头部朝向目标                                        | true          |
| immediately |         | 是否没有视角转动动画 | false         |

示例
--------

    CreepyStare:
      Skills:
      - look{headOnly=true;immediately=true} @Target

看向目标且仅限头部转动,无转动动画
