技能: Set Rotation
--------------------------

设置目标朝向.

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| relative  |         | 修改后的朝向角度 | 无  |
| yaw       |         | 修改后的视角水平旋转角度                                      | 0             |
| pitch     |         | 修改后的视角俯仰视角度                                    | 0             |

示例
--------

      Skills:
      - setrotation{relative=true;pitch=-45}
