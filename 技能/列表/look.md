技能: Look
--------------------------

看向技能目标  
除**盔甲架**外, 实体 **必须** 拥有AI（这意味着此技能效果会与默认AI"对峙"）.  
解决办法: 修改AI行动器(/实体/AI)

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| headOnly    | ho | 是否只有头部朝向目标                                        | false |
| immediately | immediate, i | 是否没有视角转动动画 | true |
| force | f |  | false |
| forceppaper | fp |  | false |

示例
--------

```yaml
 Skills:
 - look{headOnly=true;immediately=true} @Target
```
看向目标且仅限头部转动,无转动动画
