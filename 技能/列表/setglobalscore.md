技能: Set Global Score
--------------------------

设置虚拟玩家: __GLOBAL__ 在指定记分板上的分数.

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| objective | obj, o  | 用于修改的记分板名,不存在则创建 | 无 |
| value     | v       | 修改值 | 0 |
                                                                            
示例
----

    - setglobalscore
        {
        objective=someobjective;
        v=2
        } ~onAttack
