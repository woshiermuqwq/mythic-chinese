技能: Set Name
--------------------------

设置目标(非玩家)的显示名.

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| setname | 无 | 修改后的显示名
 | 无 |

示例 (实体配置)
--------

      Skills:
      - setname{name=<target.name>} @self ~onDamaged

受伤后将自身的显示名修改为 当前目标 的名称.

额外信息
-------

**支持** [占位符](/技能/占位符)与[变量](/技能/变量)（≥4.9）
