技能: Pose Armor Stand
--------------------------

修改技能目标（盔甲架）的姿势

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| head      |         | 头部姿势 | 0,0,0         |
| body      |         | 身体姿势 | 0,0,0         |
| leftarm   |         | 左手姿势 | 0,0,0         |
| rightarm  |         | 右手姿势 | 0,0,0         |
| leftleg   |         | 左腿姿势 | 0,0,0         |
| rightleg  |         | 右腿姿势 | 0,0,0         |

示例
--------

```yaml
 Skills:
 - posearmorstand{head=45,0,0} @self ~onSpawn
```
施法者生成后令自身的头部绕X轴旋转45°
