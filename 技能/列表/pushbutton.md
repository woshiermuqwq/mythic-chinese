技能: Push Button
--------------------------

按下指定位置的按钮.

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| x         |  | X轴坐标           | 0             |
| y         |  | Y轴坐标           | 0             |
| z         |  | Z轴坐标.           | 0             |

示例
--------

```yaml
 Skills:
 - pushbutton{x=15;y=67;z=-213}
```
