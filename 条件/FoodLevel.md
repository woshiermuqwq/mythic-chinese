**描述:** 若实体的饱和度在指定数值范围内.

---

**修改项:**

| 修改项名  | 别称           | 描述                      |
| --------- | -------------- | ------------------------- |
| amount | a, food, f | 所检测的数值范围 |

---

**示例:**

```
TargetConditions:
- FoodLevel{a=<10} true

```

---

**拓展信息:**

- [x] 检测对象: 实体
- [x] 别称: hunger, food, hungerlevel
