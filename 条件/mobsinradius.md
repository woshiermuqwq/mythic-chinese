**描述:** 若指定范围内的实体数量在指定范围内.

---

**修改项:**

| 修改项名  | 别称           | 描述                      |
| --------- | -------------- | ------------------------- |
| types     | t     | 用于检测的实体种类(可为MythicMobs实体) |
| amount    | a     | 检测数量 |
| radius    | r     | 检测的半径范围            |

---

**示例:**

```
Conditions:
- mobsInRadius{types=NewZombie;amount=5to10;radius=15}
```

---

**拓展信息:**

- [x] 检测对象: 实体
