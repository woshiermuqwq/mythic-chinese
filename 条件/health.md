**描述:** 若实体当前生命在指定数值范围内.

---

**修改项:**

| 修改项名  | 别称           | 描述                      |
| --------- | -------------- | ------------------------- |
| amount    | a       | 用于检测的血量数值范围 |

---

**示例:**

```
Conditions:
- health{h=50} true
```

```
# 当前生命在50以下
Conditions:
- health{h=<50} true
```

```
# 当前生命在10以上
Conditions:
- health{h=>10} true
```

---

**拓展信息:**

- [x] 检测对象: 实体
- [x] 别称: hp
