技能: Teleport
--------------------------

传送至技能目标的位置（可跨世界）

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| spreadh   | sh      | 水平最大偏移值 | 0             |
| spreadv   | sv      | 垂直最大偏移值   | 0             |

示例（实体配置）
--------

```yaml
传送测试:
 Type: husk
 Skills:
 - teleport{spreadh=5;spreadv=0} @self ~onDamaged
```
受伤后将传送到自身位置,最大水平偏移值为5格方块.
