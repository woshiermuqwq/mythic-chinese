技能: Set Mob Color
--------------------------

设置技能目标（特定实体）的外观颜色.

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| color | c | 修改后的颜色名 | WHITE |

示例  (实体配置)
--------
    
```yaml
 Skills:
 - setcolor{color=blue} @self ~onSpawn
```
生成后修改自身外观颜色为蓝色

额外信息
--

- [x] 别称: setcolor
