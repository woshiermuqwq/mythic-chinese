**描述:** 若所处区块内的实体数量在指定范围内.

---

**修改项:**

| 修改项名  | 别称           | 描述                      |
| --------- | -------------- | ------------------------- |
| amount    | a     | 检测数量 |

---

**示例:**

```
Conditions:
- mobsinchunk{a=1to5} true
```

```
Conditions:
- mobsinchunk{a=<5} true
```

---

**拓展信息:**

- [x] 检测对象: 实体
