技能: Set Mob Score
--------------------------

设置技能目标在指定记分板上的分数（上限`2147483647`, 下限为`-2147483647`）.

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| objective | obj, o  | 用于修改的记分板名,不存在则创建 | 无 |
| value     | v       | 修改值 | 无 |             
                                                                            
示例
----

```yaml
 Skills:
 - settargetscore{o=se;v=2} @self ~onAttack
···
造成伤害后将自身在记分板: sE 的分数设为2.
