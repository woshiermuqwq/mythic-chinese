**描述:** 若技能目标（玩家）背包内指定槽位上的物品是否与被比较的物品相似.

---

**修改项:**

| 修改项名  | 别称           | 描述                      | 默认值 |
| --------- | -------------- | ------------------------- | --- |
| slot | s | 用于检测的槽位 | HAND |
| item | i, material, m, mm, mythicitem | 用于检测的物品, 支持Mythic物品（支持[占位符](/技能/占位符)与[变量](/技能/变量)） | 无 |

新增于 MM 4.14.0

---

**示例:**

```
Conditions:
- itemissimilar{i=测试物品;slot=25} true


**拓展信息:**

- [x] 检测对象: 实体
- [x] 别称: issimilar, similarto
