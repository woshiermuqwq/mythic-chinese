技能: Random Message
--------------------------

随机发送多条聊天栏信息中的一条给技能目标(玩家).

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| messages  | message, msgs, msg, m       | 用于发送的多条聊天栏信息 | 无 |

示例
--------

```yaml
 Skills:
 - randommessage{
  m=
  "阿巴阿巴阿巴",
  "叭叭叭叭",
  "abab";
  } @PIR{r=20} ~onInteract
```或
```yaml
 Skills:
 - randommessage{m="阿巴阿巴阿巴","叭叭叭叭","abab";} @PIR{r=20} ~onInteract
```
俩种格式共同效果为: 施法者被右键后向20格方块范围内的所有玩家发送多则消息: `阿巴阿巴阿巴`, `叭叭叭叭`, `abab` 中的一条

额外信息
--

- [x] 别称: randommsg, rmsg, rm
