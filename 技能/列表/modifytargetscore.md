技能: Modify Target Score
--------------------------

修改技能目标指定记分板上的分数（上限2147483647, 下限-2147483647）.

可用修改模式:

-   SET(设置)
-   ADD(加)
-   SUBTRACT(减)
-   MULTIPLY(乘以)
-   DIVIDE(除)
-   MOD(除以后取余)

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| objective | obj, o  | 用于修改的记分板名,不存在则创建 |         |
| action    | a       | 修改模式                                                                                                         | ADD     |
| value     | v       | 修改值                                                                                          |         |

示例
--------

```yaml
 Skills:
 - modifytargetscore{o=测试;v=1;a=SET} @self ~onInteract
```
右键后将自身在记分板: `测试` 上的分数设为1

额外信息
---

- **支持** 占位符
- 缩写: mts
