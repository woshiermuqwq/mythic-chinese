**描述:** 若技能目标(玩家)的原版等级在指定数值范围内.

---

**修改项:**

| 修改项名  | 别称           | 描述                      |
| --------- | -------------- | ------------------------- |
| level     | l     | 用于检测的原版等级范围或具体值 |



示例
---

```
Conditions:
- level{l=10} true
```

```
Conditions:
- level{l=>10} true
```

```
Conditions:
- level{l=1to10} true
```

**拓展信息:**

- [x] 检测对象: 实体
