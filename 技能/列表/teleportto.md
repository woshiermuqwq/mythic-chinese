技能: Teleport
--------------------------

将目标传送到指定位置.

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| location, coordinates | loc, l, c | 目标传送后的所处位置坐标                     |               |
| world                 | w         | 目标传送后的所处世界              |               |
| yaw                   | y         | 目标传送后视角俯仰视角度                   | 0             |
| pitch                 | p         | 目标传送后视角水平旋转角度                      | 0             |
| relative                 | r        | 坐标是否为相对于施法者的坐标(相当于偏移),而不是相对于世界的坐标 | false              |
| origin                | o         | 坐标原点是否为施法者或目标 | 无 |

示例
--------

    传送测试:
      Skills:
      - teleportto{location=190,64,200} @PIR{r=50}
