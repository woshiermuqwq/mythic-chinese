**描述:** 若技能目标数在指定数值范围内.

修改项
---

| 修改项名  | 别称           | 描述                      | 默认值 |
| --------- | -------------- | ------------------------- | - |
| amount | a | 所检测的技能目标数量范围 | >0 |

示例
---

```yaml
 Conditions:
 - targets{a=>0} true
```
```yaml
 Conditions:
 - targets{a=0to5} true
```

拓展信息
---

- 检测对象: 技能
