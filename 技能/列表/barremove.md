技能: Bar Remove
--------------------------

为技能目标移除Boss血条.  
多条Boss血条内部ID一致时, 只移除最近一次创建的Boss血条

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| name      | n       | 要修改的血条内部ID（支持[占位符](/技能/占位符)）.                                | infobar                     |

示例 (实体配置)
--------

```
    Skills:
    - barRemove{name="测试"} @self ~onInteract
    - ...
```
