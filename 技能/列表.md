技能列表
===============

技能可写于技能配置内(技能配置位于实体配置内)  
技能也可写于技能组配置内(技能组配置位于Skills文件夹内)  
少数技能无需[目标选择器](/技能/目标选择器).

特殊技能
-----------------------

一些目标选择器作用于特殊技能,效果将有所变化(如@origin).  
特殊技能几乎全都可以调用其它技能组.  
有许多其它技能没有的模块.

| 技能名            | 描述                                                                             |
|---------------------|-----------------------------------------------------------------------------------------|
| **[Skill][]**       | 向目标激活指定技能组                                       |
| [Aura][]            | 给予目标光环 |
| [CancelEvent][]     | 取消对应的触发器事件                                  |
| [Cast][]            | 向目标激活指定技能组(更多修改项)                                     |
| [Chain][]           | 在多个目标之间连锁激活技能                       |
| [ChainMissile][]    | 在多个目标之间连锁激活导弹(Missile)                |
| [Delay][]           | 延迟                                              |
| [Global Cooldown][] | 设置施法者所有技能的冷却时间                                                 |
| [Missile][]         | 向目标发射追踪抛射物                                                       |
| [Modify Projectile][] | 修改抛射物                                   |
| [onAttack][]        | 给予目标光环(可提供伤害加成)                 |
| [onDamaged][]       | 给予目标光环(可提供伤害减免加成)            |
| [onShoot][]         | 给予目标光环(目标射击时触发效果)            |
| [Orbital][]         | 生成抛射物环环绕目标                 |
| [Projectile][]      | 向目标发射抛射物                               |
| [Shoot][]           | 向目标发射弹射物                                                  |
| [Volley][]            | 向目标发射弹射物(更多I修改项) |
| [SudoSkill][]         | 令目标作为施法者激活技能                                       |
| [Random Skill][]      | 向目标激活多个技能组内的一个                                    |
| [Totem][]             | 图腾(有严重漏洞)       |
| [Variable Add][]      | 增加变量值                                   |
| [Variable Math][]     | 计算变量值                                    |
| [Set Variable][]      | 创建变量值(若存在则替换)                                           |
| [Variable Unset][]    | 移除变量值                                                   |
| [Variable Subtract][] | 减少变量值                            |

通用修改项
--------------------

这些修改项可用于所有技能  
20tick=1秒.

| 修改项名      | 缩写 | 描述                                    | 默认值 |
|----------------|-----------|------------------------------------------------|---------|
| cooldown       | cd        | 冷却时间(秒)                                     | 0       |
| delay          |           | 延迟(刻)           | 0       |
| repeat         |           | 重复激活次数 | 0       |
| repeatInterval |           | 重复激活间隔(刻) | 0       |
