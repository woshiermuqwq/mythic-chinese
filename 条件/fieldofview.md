**描述:** 若实体处在施法者的指定视野范围之内.

---

**修改项:**

| 修改项名  | 别称           | 描述                      |
| --------- | -------------- | ------------------------- |
| angle     | a     | 用于检测的视野度数 |
| rotation  | r     | 视野起始角度     |


---

**示例:**

```
TargetConditions:
- fieldofview{angle=90} true
```

---

**拓展信息:**

- [x] 检测对象: 实体
- [x] 别称:
