技能: Set Leash Holder （5.0.5）
--------------------------

将施法者牵着的栓绳递给技能目标.

修改项
----------

| 修改项名 | 别称    | 描述 | 默认值 |
| - | - | - | - |
| remove | r, boolean, bool, b | 是否移除施法者手中的栓绳 | false |

示例
--------

```yaml
 Skills;
 - setleashholder @trigger ~onattack
```
攻击后将手中的栓绳递给受击者

额外信息
-------

- [x] 别称: setleasher
