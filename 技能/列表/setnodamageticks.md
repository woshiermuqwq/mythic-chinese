技能: Set No Damage Ticks（4.11）
--------------------------

设置技能目标的伤害硬直

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| ticks     | t, duration, d | 修改后的受伤间隔(刻) | 0 |

示例 (实体配置)
--------

```yaml
 Skills:
 - setNoDamageTicks{ticks=0} @trigger ~onAttack
```
施法者造成伤害后将受击者受伤间隔设为0刻.

额外信息
--

- [x] 别称: setimmunityticks
