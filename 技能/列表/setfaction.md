技能: Set Faction
--------------------------

修改技能目标所处的阵营.

修改项
----------

| 修改项名 | 别称    | 描述                                                                                                    | 默认值 |
|-----------|------------|----------------------------------------------------------------------------------------------------------------|---------------|
| faction | f | 技能目标应处在的阵营名称 | 无 |

示例（实体配置）
--------

```yaml
 Skills:
 - setfaction{f=cs} @self ~onSpawn
```
生成后将自身加入阵营: `cs`.
